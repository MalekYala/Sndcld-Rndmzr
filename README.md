# Sndcld Rndmzr

Great way to find 2 hour long mixes of any genre.

**Updated for 2025** - Now uses the latest SoundCloud API and widget embed system.

## Features
- Random track discovery from SoundCloud
- Filter by genre, duration, BPM, tags, and more
- Direct playback using SoundCloud's widget player
- Clean, responsive interface

## Demo
DEMO: http://malekyala.github.io/Sndcld-Rndmzr/

## How it Works
This application uses:
- SoundCloud API v2 with a valid client ID
- HTTPS endpoints for secure communication
- SoundCloud's widget player for reliable playback
- Backbone.js for application structure

## Development
Simply open `index.html` in a modern web browser. No build process required - this is a static web application.

## Technologies
- Backbone.js for MVC structure
- jQuery for DOM manipulation
- SoundCloud API for track discovery
- Pure CSS for styling

## Notes
- The SoundCloud API client ID included is a public demo key
- For production use, register your own app at https://soundcloud.com/you/apps
- The application makes client-side API calls directly to SoundCloud
